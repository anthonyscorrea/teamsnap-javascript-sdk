## `loadDivisionStore(params, callback)`
Loads a singular `divisionStore` item based on given params.

### Params
* `params`: [int] - an object with query parameters.
* `callback`: [function] - callback to be executed when the operation completes.

### Examples
```javascript
// ~~~~~
// Loads `divisionStore` for `divisionId: 1`.
teamsnap.loadDivisionStore({divisionId: 1});
